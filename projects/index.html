Lemonade Ledger

This application is a custom built point-of-sale application for Charleston, SC-based retail company Lowcountry Lemonade. It tracks sales and inventory, allowing the business owner to monitor multiple locations from their smart phone in real time.

This application was built using Ruby on Rails, with the main point-sale-page built with pure JavaScript.

Check out the source code on Github.

Teachable

Teachable is a platform to organize curricular materials for school teachers. It is built using Ruby on Rails with a large helping of JavaScript and jQuery to make the front end interactive. The PostgreSQL database is queried mainly via the Rails ORM.

Check out the source code on Bitbucket.
